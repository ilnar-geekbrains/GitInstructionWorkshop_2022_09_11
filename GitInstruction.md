## Ветки. Локальная работа с ветками в git.

Создание новой ветки **git branch <name_branch>** 

Чтобы перейти новую ветку **git checkout <name_branch>**

Для того чтобы слить данные с разных веток

Для начала переходим в ветку, в которую хотим добавить данные

    **git merge <name_branch>** в name_branch пишем имя ветки из которой хотим добавить данные

Для удаления ненужной ветки **git branch -d <name_branch>**
