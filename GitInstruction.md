### Базовая работа с ветками в git.

#### Вывод списка веток

Для вывода списка имеющихся веток необходимо использовать команду:

    git branch

В списке будет указано на какой ветке мы сейчас находимся - знаком __*__ и имя ветки будет подсвечено зелёным цветом.

#### Создание новой/другой ветки

Для создания новой ветки необходимо использовать команду:

    git branch <branch_name>

где, <branch_name> - имя ветки.

#### Переход на другую ветку

Для перехода на другую ветку необходимо использовать команду:

    git checkout <branch_name>

Для возврата на основную ветку необходимо использовать команду:

    git checkout master

#### Слияние веток

Слияние веток происходит в текущую из той, которую запрашиваем, используя команду:

    git merge <branch_name>

В этом случае в тукещую ветку (например, **master**) сольётся информация из ветки **<branch_name>**.

#### Удаление веток

Для удаления какой-либо ветки необходимо использовать команду:

    git branch -d <branch_name>

указав имя конкретной ветки, которую требуется удалить.

#### Открытие древа изменений

Древо изменений возможно вызвать несколькими способами, при помощи команд:

    git log --graph

при этом отобразится древо с расширенной информацией всех **commit** текущей ветки;

    git log --oneline --graph

при этом отобразится древо всех **commit** текущей ветки в сокращенном виде **commit** (семь символов идентификатора **commit** и комментарий к нему);

    git log --oneline --all --graph
