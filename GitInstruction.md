### Инструкция по созданию pull request

1. Делаем форк (fork) интересующего репозитория
2. Делаем git clone
3. Создаем ветку
4. Производим изменения в этой ветке
5. Отправляем измененияв свой аккаунт (push)
6. Делаем pull request в github
