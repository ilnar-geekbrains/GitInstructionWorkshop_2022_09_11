### Базовая работа с локальным репозиторием
Занесение изменений в репозиторий производится с помощью нескольких команд:
* `git add` - позволяет перенести изменения в файле из рабочего файла в раздел проиндексированных файлов;
* `git commit` - фиксирует проиндексированные действия;
* `git status` - с помощью этой команды можно получить текущее состояние git, какие файлы были изменены;
* `git diff` - показывает разницу между рабочей копией и проиндексированным файлом.
