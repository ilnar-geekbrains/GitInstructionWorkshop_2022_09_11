## Ветки. Локальная работа с ветками в git.

Для создания новой ветки нужно ввести в терминал **git branch <имя_новой_ветки>**

Для просмотра всех существующих веток нужно ввести **git branch**

Для перехода на другую ветку нужно ввести **git checkout <название_ветки>**

Для присоединения ветки к текущей нужно ввести **git merge <название_ветки>**

Для удаления ветки нужно ввести **git branch -d <название_ветки>**
